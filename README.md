Конкретно Объе́ктно-ориенти́рованное программи́рование (сокр. КООП) (specifically object-oriented programming (SOOP)) — методология или стиль программирования на основе описания типов/моделей предметной области и их взаимодействия, представленных порождением из прототипов или как экземпляры классов. 

Из предметной области вычленяется сущность, которая описывается программным кодом в отдельном от других описанных сущностей месте. Этим местом может быть директория, файл, класс, структура.

Отличается от ООП тем, что не обязательны: наследование, слабо связанные между собой модули.

Обьединять функции и состояние сущности не обязательно в одном классе, может быть достаточным объединение структуры и функций в одном файле.
